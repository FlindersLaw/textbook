## Ex1 Exercises
This first set of exercises introduces you to the basics of coding with Docassemble.
You'll learn how to:

* ask questions and display answers;
* format text;
* generate Word and PDF documents; and
* use logic to control execution of your app.

These exercises are designed to gradually introduce you to coding. Start with 
the first and work your way through the list.  Watch the recommended videos before 
you start each exercise.
