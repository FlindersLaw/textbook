In our [Hello World 4 example](/docs/textbook/chapter_02/00-42) in <DisplayChapter chapter='chapter_02'/>
we used <DisplayGlossaryItem item='markdown' upper /> tables to display information in a table in our
final screen.  A [`table` block](/docs/textbook/chapter_02/00-42) is an alternative way to display information
in a table.  Table blocks work best for when we have data already structured in a complex data structure,
usually a list or list of dictionaries.

For a more detailed example of table blocks, and for an example of how they are used, refer to the
[Filtering and Advanced Iteration](/docs/textbook/chapter_06/filtering) case study and in particular
[here](/docs/textbook/chapter_06/filtering/00-40#about-table-blocks).
