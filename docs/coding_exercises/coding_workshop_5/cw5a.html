Exercise 5a

References:
  • List Length
  • Markdown Tables

Background​

In this exercise we're going to take the code from Exercise 4c from Coding Workshop 4 and build on it so that we ask followup questions.

We're going to ask two questions and we're going to implement them in two different ways. Both ways are equally valid and have their (slight) advantages and disadvantages, as we'll discuss in class.

Whiskers on Kittens​

It may be that many people like whiskers but not on kittens. We're going to add some other animals on which people might like whiskers. If people select 'Whiskers on Kittens' they'll be asked a followup question confirming which animal they really like whiskers on.

We'll implement this followup question using a nested if/elif statement, much like how we did in Coding Workshop 3.

Warm Woolen Mittens​

Again, we want to give people choice. If a user selects 'Warm Woolen Mittens' we want to ask a followup question so we can determine what colour they'd prefer their warm woolen mittens to be.

We'll implement this followup question directly in the t_mittens template.

Steps​

  1. Make a new Playground Project called CodingWorkshop5
  2. Copy the example code below into files in your CodingWorkshop5 playground. The file names must match exactly.
  3. Complete the code in cw4c_code.yml so that templates are appended to temp for each animal in the whiskers_followup question.
  4. Add a Mako if statement to the t_mittens template so that appropriately coloured mittens are displayed depending on the value of mitten_colour.

cw5a_code_exercise.yml

---
code: |
  temp = []
  # We iterate over all of the favourite things
  for item in favourite_things.keys():
    # First we check if this favourite thing is selected
    if favourite_things[item]:
      # If it is selected we add the appropriate template to temp
      if item == 'a_raindrops':
        temp.append(t_raindrops)

      elif item == 'b_kittens':
        # We need to process our followup question here
        # Use an if/elif statement to process each possible answer
        # to whiskers_followup and append the appropriate template
        # (the templates have been completed for you)
        if whiskers_followup == 'kittens':
          temp.append(t_kittens)
        # the elif statements go in here

        # This is our catchall in case we get an error 
        else:
          error_subject = 'Unknown answer'
          error_content = 'Do not have a template for `whiskers_followup`: `{}`'.format(whiskers_followup)
          temp.append(t_error)

      # Now we get onto the rest of the favourite things elifs
      elif item == 'c_kettles':
        temp.append(t_kettles)
      elif item == 'd_mittens':
        temp.append(t_mittens)
      elif item == 'e_packages':
        temp.append(t_packages)
      else:
        error_subject = 'Unknown item'
        error_content = 'Do not have a template for item: `{}`'.format(item)
        temp.append(t_error)

  template_list = temp
