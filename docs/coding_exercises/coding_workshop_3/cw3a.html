Exercise 3a

References:
  • Structuring Code
  • Minimising Mandatory Blocks

This exercise is about setting ourselves up so that we can get organised. We will to use good coding structure, multiple files and a separate project to keep ourselves organised:

  1. Create a new Project in your Playground. Call it Ex3a.
  2. In your new project create these files:
    • main.yml
    • questions.yml
    • templates.yml
    • screens.yml
    • code.yml
  3. In your main.yml add an include block which includes all these files (except main.yml!)
  4. Create a mandatory question block as a welcome screen. Add some basic explanatory text that describes what this app is going to do.
  5. Create a mandatory code block underneath your question block. The code block should just contain a pass command in it.
