Exercise 2a

References:
  • Logic Tests
  • Minimising Mandatory Blocks
  • Chapter Four

We are going to create this part of the app:

We need to make:

  • the welcome screen
  • A screen to ask the user what pet they own
  • An exit screen for users that don't own a dog or cat
  • some code to drive our app

Welcome Screen​

Let's start with a mandatory question block that displays a welcome screen explaining what this app does. You may also want to include a disclaimer stating that this app does not provide legal advice.

Question Screen​

Create a question block that asks the user what kind of pet they have. They should be able to choose from this list:

  • Cat
  • Dog
  • Fish
  • Crocodile
  • Something else

Use radio buttons to present the choices. Also, use key/value pairs to help in making the names easier for the user to read your code easier to process eg:

...
fields:
  - What is your pet?: W2pet
    choices:
      - Dog: dog
      - Cat: cat
      ...
      - Something else: other
---
