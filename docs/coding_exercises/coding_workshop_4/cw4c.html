Exercise 4c

References:
  • If Statements
  • Nested If Statements
  • Templates

Background​

In this exercise we are going to select templates for each selected favourite thing.

We are going to build on our code from Exercise 4b and display text from templates instead of our table.

Because our user can choose more than one favourite thing we may need to display more than one template on our final screen. Sounds like a job for iteration!!

Now that we're introducing templates our code is going to get rather big, so we'll also split our code out into multiple files.

Your program should display something like this

Steps​

  1. Copy the example code below into files in your CodingWorkshop4 playground
  2. Complete the code in cw4c_code.yml so that all the templates are part of the if/elif statement.
  3. Update final_screen to display template_list
note

You must use exactly the same file names, otherwise the includes will break.

hints

Completing the code block​

All the templates have been completed for you and are in templates.yml. You don't need to create these. Just make sure each of the five templates forms part of the if/elif statement in cw4c_code.yml.

Displaying the selected templates​

The code block adds a template to template_list only if the relevant checkbox was checked. In our final screen we need to display each of the templates listed in template_list.

You will notice that each template has a subject as well as content. We can use the subject as a heading above the content. To display a template's subject we do something like this (for example):

  t_kettles.subject

We will use iteration to display each template in our template_list. The % for loop line will look something like this:

  % for item in template_list:

Therefore, to display each template's subject we can use:

  ${item.subject}

and to display each template's content we can use:

  ${item}

(as the content is always displayed by default)

cw4c_code.yml

# Coding Exercise 4c
#
# Displaying a list of templates
---
code: |
  temp = []
  # We iterate over all of the favourite things
  for item in favourite_things:
    # First we check if this favourite thing is selected
    if favourite_things[item]:
      # If it is selected we add the appropriate template to temp
      if item == 'a_raindrops':
        temp.append(t_raindrops)
      elif item == 'b_kittens':
        temp.append(t_kittens)
      elif item == 'c_kettles':
        temp.append(t_kettles)
      elif item == 'd_mittens':
        temp.append(t_mittens)
      elif item == 'e_packages':
        temp.append(t_packages)
      else:
        error_subject = 'Unknown item'
        error_content = 'Do not have a template for item: `{}`'.format(item)
        temp.append(t_error)

  # Always assigne the temp variable to the 'real' variable last. 
  # This guarantees that Docassemble always runs the entire code block
  template_list = temp
